# Event site

  • Media Query - 각각의 디바이스에 맞는 화면 구성
  • media query - Cross Browsing - 웹호환성 웹접근성

# ResPonsive

# Corporate site

# Summary blog

  • FTP - 클라이언트 서버를 사용한 프로젝트 정리 관리
